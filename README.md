# MIRAI 🚀

MIRAI is a Streamlit-powered Python application featuring a **Multiverse of Chatbots** and a built-in **Calculator** utility. 

## 🌟 Features

*   **Multiverse of Chatbots:** Interact with multiple AI chatbot personalities or models seamlessly within a single interface.
*   **Streamlit Calculator:** A clean, interactive calculator utility built natively with Streamlit components.
*   **User-Friendly Interface:** Clean navigation and responsive design powered by Streamlit.
